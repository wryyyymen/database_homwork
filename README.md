# 医院门诊挂号系统

## 项目介绍

本项目是一个基于 Java Swing + MySQL + JDBC 的医院门诊挂号系统，主要用于课程设计或 Java GUI 项目练习。

## 技术栈

- Java 17
- Swing
- JDBC
- MySQL
- Maven

## 功能模块

- 管理员登录
- 科室管理
- 医生管理
- 排班管理
- 患者管理
- 挂号预约
- 缴费管理
- 就诊记录管理

## 运行环境

- JDK 17 或以上
- MySQL 8.0 或以上
- IntelliJ IDEA
- Maven
